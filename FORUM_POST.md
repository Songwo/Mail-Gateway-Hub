## 标题：[开源] 别再被垃圾邮件轰炸了！自建 AI 邮件网关：Gemini 摘要 + 飞书/Gotify 实时推送

各位 L 站的大佬们好！

作为一名合格的极客，我的邮箱每天都会塞满各种验证码、监控告警、Github 通知和（删不完的）营销邮件。
为了拯救我的通知栏，我撸了一个 **Mail-Gateway-Hub**。

### 💡 它能做什么？
- **不仅是通知**：它调用 Gemini 1.5 Flash 帮你读信。
- **秒读重点**：1000字的周报变 50 字摘要；验证码直接高亮提取。
- **颜值即正义**：基于飞书交互式卡片，重要邮件红色预警。
- **全异步监听**：基于 IMAP IDLE，新邮件到，手机秒响。

### 🛠️ 技术栈
Python 3.10 + imap_tools + Gemini API + Docker
